# 💰 Personal Budget Tracker
A Python-based Personal Budget Tracker used to manage income and expenses. The application allows users to record daily transactions, categorize spending, calculate remaining balance, and view financial summaries. Built using Python, file handling, and JSON for data storage.

A simple and beginner-friendly Python application to manage personal income and expenses.

---

## 📌 Features

✅ Add Income  
✅ Add Expenses  
✅ Categorize Spending  
✅ View Budget Summary  
✅ Calculate Remaining Balance  
✅ Store Data Using JSON  

---

## 🛠️ Technologies Used

- Python
- JSON
- File Handling
